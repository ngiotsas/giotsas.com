Why We Used CSS Sprites in 2024

When building this website, we had to decide how to handle our icons. Everyone talks about SVGs and icon fonts, but we went with CSS sprites. Here's why.

The Decision

As beginners, we wanted to understand the fundamentals before jumping to modern solutions. CSS sprites taught us about background positioning, image optimization, and how browsers handle resources.

What We Learned

Creating our sprite sheets manually taught us about:

  • Image compression and optimization
  • CSS background positioning
  • Browser caching strategies
  • HTTP request reduction

Sure, it's more work than using an icon library, but we understand exactly how our icons work and why they're efficient.

March 15, 2024 Python

Our First Month with Python

A month ago, we started learning Python. Here's an honest look at what we've learned, what confused us, and what finally clicked.

Week 1: Hello World and Variables

Python's syntax felt weird coming from other languages. No semicolons? Indentation matters? But after a few days, we started appreciating how clean it looks.

Week 2: Functions and Loops

This is where things got interesting. Python's for loops are so much cleaner than other languages. The range() function confused us at first, but now we use it constantly.

Week 3: File Handling

Built our file organizer script this week. First time working with actual files in Python. The with statement for file handling is brilliant once you understand it.

Week 4: Libraries and Modules

Started exploring the Python Standard Library. So many useful modules! Currently playing with datetime, os, and shutil for our automation scripts.

March 10, 2024 Learning

Building This Website: What We Learned

Creating giotsas.com was our biggest web development project yet. Here's what we learned during the process.

Planning Phase

We started by sketching layouts on paper. Wanted something that looked professional but reflected our personality as learners, not experts.

HTML Structure

Focused on semantic HTML from the start. Used proper heading hierarchy, meaningful class names, and structured content logically.

CSS Challenges

Getting the sprite positioning right took forever. Also learned about CSS Grid the hard way - lots of trial and error with grid-template-areas.

JavaScript Additions

Added search functionality and form handling. First time working with DOM manipulation extensively. The search feature was particularly challenging.

What We'd Do Differently

Next time we'd start with mobile-first design. We built desktop-first and had to rework several media queries. Also would have planned our CSS better - some of our styles got messy.

March 5, 2024 Tools

Our Development Setup

People always ask what tools we use for coding. Here's our current setup as beginner developers.

Code Editor

VS Code with Python and HTML extensions. The integrated terminal is incredibly useful, and the syntax highlighting helps catch errors early.

Browser Testing

Chrome for development (DevTools are amazing), but we test in Firefox and Safari too. Mobile testing mostly on Chrome's device simulation.

Version Control

Git and GitHub. Still learning proper commit messages and branching, but we backup everything now after losing code once.

Learning Resources

MDN for web development reference, Python.org docs for Python, Stack Overflow when we're stuck, and YouTube for visual explanations.
